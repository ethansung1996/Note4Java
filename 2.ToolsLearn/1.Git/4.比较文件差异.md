# 4.比较文件差异

实际工作中，我们需要对比我们同一个文件在不同版本时候的差异，就需要对比这个功能，也有可能本地库和工作区对比，本地库和暂存器，暂存器和工作区




|       命令        |     比较区域      |
| ----------------- | --------------- |
| git diff          | 工作区 vs 暂存区 |
| git diff HEAD     | 工作区 vs 版本库 |
| git diff --cached | 暂存区 vs 版本库 |
