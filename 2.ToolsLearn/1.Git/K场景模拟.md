## 比较前一个版本和新版本的差异
>今天你把新的更新从远程pull下来，你想看看之前的版本和今天刚pull下来的版本有什么差异，也就是更新了什么?
```
git diff HEAD^ HEAD
```
这就会显示出来新旧版本的差异，新版本更新了什么


理解：最好是`git diff 老版本 新版本`老版本在前，新版本在后。这是两个版本库的比较，是我们常用的比较方式，比较好理解新增和删除；
